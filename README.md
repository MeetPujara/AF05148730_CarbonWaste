# 🌎 Waste Management Application
The Waste Management Application is a comprehensive software solution designed to manage and track waste data, providing valuable insights and calculations to support environmental sustainability. This application aims to simplify the process of collecting, storing, and analyzing waste data, making it easier for organizations. The core features of this application include data storage, calculation, and analysis, all of which are integrated into console based app.

- **Calculation**: The application performs calculations related to waste, such as waste quantities, emissions, and environmental impact metrics.

## 🛠️ Tech Stack
* Java
* MySQL
* JDBC
